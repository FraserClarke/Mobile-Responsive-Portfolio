# Mobile-Responsive-Portfolio
A Mobile Responsive Portfolio utilizing "Bootstrap" 

## Description

The aim was to create a mobile responisve portfolio website including three
pages which incorporates: 
* symantic html elements, 
* bootstrap, 
* working links, 
* consistent navbar,
* Responisve resizing
